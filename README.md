# Luciano Wash Limpieza Profesional

Web-app/PWA para Luciano Wash como division de limpieza profesional de LPS Company Group.

## Abrir local

Abre `index.html` en el navegador o levanta un servidor local:

```powershell
python -m http.server 8080
```

Luego visita `http://localhost:8080`.
