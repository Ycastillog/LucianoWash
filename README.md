# Luciano Wash Limpieza Profesional

Primera version de web-app/PWA para Luciano Wash como empresa de limpieza profesional integral.

## Abrir local

Abre `index.html` en el navegador o levanta un servidor local:

```powershell
python -m http.server 8080
```

Luego visita `http://localhost:8080`.

## Pendientes para produccion

- Cambiar `WHATSAPP_NUMBER` en `app.js` por el numero oficial.
- Ajustar precios reales.
- Reemplazar o ampliar la galeria con fotos reales de autos, muebles, alfombras, piscinas, hogares y comercios.
- Sustituir `assets/brand-complete.svg` por un logo final profesional cuando este disponible.
- Agregar panel administrativo cuando ya este validado el flujo de reservas.
